Análisis Inteligente del Mercado Laboral Chileno

Este proyecto tiene como objetivo construir una plataforma de análisis del mercado laboral chileno, orientada a identificar tendencias de empleabilidad, habilidades demandadas y evolución de roles tecnológicos, utilizando técnicas de análisis de datos, ingeniería de datos y, en etapas posteriores, NLP y modelado predictivo.

El enfoque del proyecto es end-to-end, abarcando desde la ingesta de datos, su almacenamiento estructurado, hasta la preparación para análisis exploratorio y generación de insights accionables.
